## 클래스와 객체(1)
### 클래스 선언과 객체 정의
#### 클래스 선언과 객체 정의
  - 객체란?
    - 소프트웨어 시스템 안의 어떠한 대상을 표현 한 것
    - 정해진 처리를 수행할 수 있음 => 행위, 메소드, 맴버함수
    - 처리과정에 따라 내부 상태가 변화할 수 있음 => 속성, 데이터 맴버
    - 다른 객체와 상호작용 할 수 있음 => 메시지 전달(맴버함수 호출)
  -  클래스란?
    - 객체의 설계도
    - 객체가 포함할 **데이터 맴버**의 선언과 **맴버함수의 정의**를 포함함
